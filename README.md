
**Malayalam Hate Speech Detection**

Developed a deep learning–based NLP system to classify abusive and non-abusive Malayalam text, addressing the challenges of low-resource languages. The model uses a fusion architecture combining transformer models like MuRIL and XLM-RoBERTa to capture both contextual and semantic features. Built using PyTorch and Hugging Face, the system includes an end-to-end pipeline for preprocessing, training, and evaluation, achieving improved performance over single-model baselines. This project demonstrates expertise in multilingual NLP, model optimization, and real-world text classification.
